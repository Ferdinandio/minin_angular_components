Обмен данными между компонентами осуществляется
при помощи декораторов @Input, @Output и EventEmmiter

  Сверху компонент с формой создаёт объект с данными. При нажатии на кнопку "Добавить пост", объект отправляется в компонент со списком постов. Список мгновенно выводит добавленный объект.

  Здесь же реализован доступ до инпута при помощи декоратора @ViewChild, для фокусировки верхнего инпута после добавления поста.

  При нажатии на кнопку "Инвертировать цвет постов", динамически добавляется или исчезает класс в компоненте со списком постов

  По условию добавляется информация о длине поста.
При помощи декоратора @ContentChild получаем добавленный по условию шаблон и выводим его в консоль.
